Privacy Policy – Portal AI

Last updated: September 2025

Data Collection

Portal AI does not collect, store, or transmit any personally identifiable information. All extension settings, including your OpenAI API key, are stored locally in your browser.

Website content is only processed when you explicitly activate the extension to summarize or analyze a page.

Use of Data

When you request a summary or ask a question, the visible text of the page is securely sent to the OpenAI API using the API key you provide. No data is transmitted to or stored by Portal AI developers.

Portal AI does not access your browsing history, credentials, or other personal information.

Third-Party Services

The extension connects only to the official OpenAI API at https://api.openai.com/*. Your usage of the OpenAI API is subject to OpenAI’s own terms and privacy policy.

Security

Your OpenAI API key is stored locally in Chrome’s extension storage. It is never sent anywhere except directly to the OpenAI API.

Contact

If you have questions about this Privacy Policy, please contact: portalmailai@gmail.com
